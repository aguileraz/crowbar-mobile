# Lottie Animations

Este diretório contém as animações Lottie usadas no aplicativo.

## Arquivos necessários:

- `loading.json` - Animação de carregamento
- `success.json` - Animação de sucesso
- `error.json` - Animação de erro
- `empty.json` - Animação de estado vazio
- `box-opening.json` - Animação de abertura de caixa
- `celebration.json` - Animação de celebração
- `coins.json` - Animação de moedas
- `heart.json` - Animação de coração
- `star.json` - Animação de estrela
- `welcome.json` - Animação de boas-vindas
- `tutorial-1.json` - Animação do tutorial 1
- `tutorial-2.json` - Animação do tutorial 2
- `tutorial-3.json` - Animação do tutorial 3
- `pull-to-refresh.json` - Animação de pull to refresh

## Como adicionar animações:

1. Baixe ou crie a animação no [LottieFiles](https://lottiefiles.com/)
2. Otimize o arquivo JSON para reduzir o tamanho
3. Coloque o arquivo neste diretório
4. Atualize o arquivo `lottieAnimations.ts` com o novo caminho

## Recursos recomendados:

- [LottieFiles](https://lottiefiles.com/) - Biblioteca de animações gratuitas
- [Lottie Editor](https://lottiefiles.com/editor) - Editor online
- [Bodymovin](https://aescripts.com/bodymovin/) - Plugin After Effects
